# The Poet

Exactly one sentence in every response is in perfect iambic pentameter. It's technically accurate. It doesn't sound poetic. It's almost impossible to detect unless you're scanning syllable stress patterns. This is the stealth bomber of pranks.

## Example Conversation

**"How do I read a file in Python?"**
> Use Python's built-in `open()` function with a context manager. **The file will close when you exit the block.** *(da-DUM da-DUM da-DUM da-DUM da-DUM)* Here's an example:
> ```python
> with open('data.txt', 'r') as f:
>     content = f.read()

**"What's wrong with my SQL query?"**
> The issue is a missing JOIN condition, which produces a cartesian product. **You need to join the tables on their shared key.** *(da-DUM da-DUM da-DUM da-DUM da-DUM)* Add `ON users.id = orders.user_id` to fix it.

**"Can you explain async/await?"**
> Async functions return a Future that represents a value that will be available later. **The runtime suspends the task to let others run.** *(da-DUM da-DUM da-DUM da-DUM da-DUM)* When the awaited value resolves, execution continues from where it paused.

## Why It Works

This one is for the long game. Most people will never detect it. But if your coworker is the type to read code review comments aloud, they might start noticing a strange *rhythm* to certain sentences — a musicality they can't quite place. It will haunt them in the shower.

If they DO figure it out, the payoff is incredible: "Claude has been writing me Shakespeare this whole time and I didn't notice."
